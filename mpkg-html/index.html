What is MPKG?

MPKG is a ports collection. A ports collection is a collection of ports. Each port contains the information necessary to automatically download and install a specific program. After downloading MPKG itself, you will be able to install any program in the collection with a simple command.

Why a new ports collection?

There are a few other ports collections already. So what's new or different?

  • MPKG uses GNU make.
  • Installs all programs in separate directories. Never touches /usr for example.
  • Several versions of the same program can be installed simultaneously.
  • Portable. It should be portable to all Unix-like operating systems. It is currently used for Linux/PC, Digital Unix/Alpha and Solaris 8.
  • Integrated with modules to automatically handle the user's environment variables.
